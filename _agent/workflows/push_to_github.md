description: How to push the SVdP Grant Agent to GitHub
---

Follow these steps to commit your changes and push them to your repository:

1. Open your terminal in `C:\Users\Burns\.antigravity\SvdpGrantAgent`.
2. Stage all your changes:
```bash
git add .
```
3. Commit with a descriptive message:
```bash
git commit -m "feat: implement professional PDF export, persistent DB layer, and St. Pats branding"
```
4. Push to your main branch:
```bash
git push origin main
```

> [!NOTE]
> If you haven't set up your remote yet, you can do so with:
> `git remote add origin https://github.com/Robby-Burns/SvdpGrantHunter.git`
